Backend zatím nemá přístup k Zásilkovně

Chybí přihlašovací údaje k API. Nastav je v Railway (nikam se nepromítnou do prohlížeče):

  1. Railway → tvůj projekt → Variables
  2. Přidej ZASILKOVNA_KEY = tvůj API klíč (Klientská sekce → Zákaznická podpora)
  3. Přidej ZASILKOVNA_PASSWORD = tvé API heslo
  4. Volitelně APP_TOKEN = libovolný tajný řetězec — tím zamkneš tyto endpointy, ať k tvým fakturám nemá přístup kdokoli s odkazem
  5. Po uložení Railway službu restartuje; pak obnov tuto stránku

Klíč ani heslo nikdy neposílej do prohlížeče — proto to čte server přes proxy.

Přístupový token

Endpointy jsou zamčené (APP_TOKEN). Zadej token, který jsi nastavil v Railway.
